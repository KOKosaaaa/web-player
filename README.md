# Web Player

Веб-плеер для аудио с визуализацией волны и редактором нарезки.

## Возможности

- **Воспроизведение** — play/pause, перемотка, громкость
- **Плейлист** — добавление, удаление, перемещение треков
- **Визуализация** — волновая форма с зумом (колёсико мыши)
- **Редактор нарезки** — выделение фрагмента, превью, применение
- **Fade Out** — плавное затухание
- **Сохранение** — треки сохраняются в браузере (IndexedDB)

## Использование

### Быстрый старт

1. Откройте `index.html` в браузере
2. Нажмите **+ Добавить** и выберите аудиофайлы
3. Кликните на трек для воспроизведения

### Управление треком

| Кнопка | Действие |
|--------|----------|
| Play/Pause | Клик на трек или кнопку воспроизведения |
| Стоп | Остановка всех треков (кнопка в шапке) |
| -10, -5, +5, +10 | Перемотка на N секунд |
| Ползунок громкости | Регулировка громкости |
| Fade | Плавное затухание и остановка |
| Reset | В начало трека |

### Редактор нарезки

1. Нажмите **Нарезка** в шапке или иконку ножниц у трека
2. Тяните зелёные ручки для выделения фрагмента
3. Используйте колёсико мыши для зума
4. Кнопки:
   - **Весь** — воспроизвести весь трек
   - **Выделение** — воспроизвести от начала выделения до конца
   - **Превью** — воспроизвести только выделенный фрагмент
   - **Стоп** — остановить воспроизведение в редакторе
   - **Создать нарезку** — обрезать трек до выделенного фрагмента
   - **Сброс** — вернуть выделение к исходным значениям

### Горячие клавиши редактора

- **Колёсико мыши** — зум волны
- **Перетаскивание** — панорамирование (при зуме > 1x)
- **Клик по волне** — перемотка к позиции

## Технические детали

- Чистый HTML/CSS/JavaScript, без зависимостей
- Поддержка форматов: MP3, WAV, OGG, FLAC, AAC, M4A, WebM
- Максимальный размер файла: 300 МБ
- Хранение в IndexedDB (данные остаются в браузере)
- Адаптивный дизайн для мобильных устройств
- Content Security Policy (CSP) для защиты от XSS
- HTML-санитизация вывода, валидация файлов при загрузке

## Установка на сервер

Просто разместите `index.html` на любом веб-сервере или откройте локально.

```bash
# Локальный сервер (Python)
python -m http.server 8080

# Открыть в браузере
# http://localhost:8080
```
